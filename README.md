# EcoImpact
A platform that encourages responsible e-waste disposal through rewards
and easy scheduling.
# Problem
Improper disposal of e-waste causes environmental damage, health risks, and loss of valuable recyclable materials.  
Most people lack awareness, incentives, and easy access to organized e-waste collection systems.
# Solution
EcoImpact makes e-waste disposal simple and seamless while motivating users through a reward-based points system, encouraging responsible recycling and reducing improper electronic waste disposal.
# How It Works
1. User signs up / logs in
2. User creates their profile
3. User schedules an e-waste collection visit
4. Visit details are stored in Firebase
5. Dashboard shows past visits, scheduled visit and reward points
# What Makes EcoImpact Unique
- Incentive-based recycling through reward points
- User-specific tracking using Firebase
- Clean, beginner-friendly UI
- Focus on awareness + action, not just information
# Tech Stack
- Frontend: HTML, Tailwind CSS, JavaScript
- Backend: Firebase Authentication & Firestore
- Tools: GitHub, Firebase Console
# How to Run
1. Clone the repository
2. Add your Firebase config in `firebase.js`
3. Open `index.html` in a browser
4. Sign up and start using EcoImpact
